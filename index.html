快速开始

三步配置，即刻接入 AI 编程服务

1

购买或兑换套餐

选择合适的服务（Claude Code / Codex），完成购买或兑换

# 登录控制台选择套餐

# 购买或输入兑换码成套餐

→
2

配置 API 地址与密钥

在控制台复制 API 地址与密钥，配置到本地/服务器环境中

export VIBE_API_BASE="https://code.claude-opus.top/" export VIBE_API_KEY="sk-xxxxx..."
→
3

开始使用

在项目中调用接口或使用命令行工具验证接入

curl -H "Authorization: Bearer $VIBE_API_KEY"

# 或多套餐程中的即时一键安装
