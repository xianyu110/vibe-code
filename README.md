## 🔧 本地开发

### 方式一：Python 服务器
```bash
git clone https://github.com/xianyu110/vibe-code.git
cd vibe-code
python -m http.server 8000
# 访问 http://localhost:8000
```

### 方式二：Node.js 服务器
```bash
npm install
npm run dev
# 或使用 npm run serve
```

### 方式三：直接打开
```bash
# 直接用浏览器打开 index.html
# 或使用 VS Code Live Server 插件
```
